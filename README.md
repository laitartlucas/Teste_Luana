# Teste de Estilo — Luana Laitart

Quiz de estilo pessoal interativo. A cliente responde 7 perguntas (texto e imagens)
e descobre os estilos que mais combinam com a sua essência, com percentual de match,
"DNA de estilo" e dicas de peças-chave.

## Arquivos
- `index.html` — o teste (abre direto no navegador)
- `logo.png` — logo da marca usada no cabeçalho (precisa ficar na mesma pasta)
- `img/` — pasta opcional para trocar os desenhos por fotos reais (veja `img/LEIA-ME.txt`)

## Como usar
Basta abrir o arquivo `index.html` no navegador.
Funciona 100% offline, sem servidor.

## Publicar online (GitHub Pages)
1. Settings → Pages → Branch `main` → Save.
2. O link ficará em `https://laitartlucas.github.io/Teste_Luana/`.

## Publicar online (Railway)
O `package.json` sobe o site com o servidor estático `serve`.
O Railway detecta o Node, roda `npm install` e `npm start` automaticamente.
